OpenAI APIキーの設定方法
=========================

【1. APIキーの取得】
1. OpenAIのウェブサイトにアクセス
   https://platform.openai.com/

2. アカウントを作成またはログイン

3. 「API Keys」ページに移動

4. 「Create new secret key」をクリック

5. 表示されたキーをコピー（sk-で始まる文字列）

【2. このシステムへの設定】
1. 「システム」フォルダを開く

2. 「.env」ファイルをメモ帳などで開く

3. 「ここにあなたのAPIキーを入力してください」の部分を
   コピーしたAPIキーに置き換える

   変更前: OPENAI_API_KEY=ここにあなたのAPIキーを入力してください
   変更後: OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxx

4. ファイルを保存して閉じる

【3. 注意事項】
- APIキーは他人に見せないでください
- GitHubなどに公開しないでください
- 無料枠を超えると課金されます

【4. 確認方法】
「議事録生成を開始」を実行して、エラーが出なければ成功です
